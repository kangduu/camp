## npm common command

### `npm view` 查看注册表信息

> **aliases** `info` `show` `v`

查看所有版本信息，并以 json 格式输出

```bash
npm view react versions --json
```

查看指定版本注册信息

```bash
npm view react@19.0.0
```

- [npm-view](https://docs.npmjs.com/cli/v10/commands/npm-view)
